## How to Contribute

- Create new topic folders under `src/pages/<section>/`
- Add `index.mdx` to each topic folder
- Update `src/pages/<section>/_meta.js` and `src/pages/_meta.js`
- Keep content interview-focused and beginner-friendly

## Goals

- Produce a complete, book-like learning path
- Cover fundamentals through advanced system architecture
- Use Python examples, diagrams, and practical design guidance
- Cross-link related topics for fast concept discovery
