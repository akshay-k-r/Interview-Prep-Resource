## Python examples

### Reverse a string in place
```python
s = list("hello")
i, j = 0, len(s) - 1
while i < j:
    s[i], s[j] = s[j], s[i]
    i += 1
    j -= 1
print("".join(s))
```

### Two-pointer subarray sum
```python
def has_pair_with_sum(nums, target):
    nums.sort()
    left, right = 0, len(nums) - 1
    while left < right:
        current = nums[left] + nums[right]
        if current == target:
            return True
        if current < target:
            left += 1
        else:
            right -= 1
    return False
```

- Prefer built-in methods for simple string operations.
