# Creational Patterns

Creational patterns focus on the mechanisms of object creation, making systems independent of how objects are composed and represented.

## Patterns

- [Singleton](/01-design-patterns/creational/singleton)
- [Factory](/01-design-patterns/creational/factory)
- [Abstract Factory](/01-design-patterns/creational/abstract-factory)
- [Builder](/01-design-patterns/creational/builder)
- [Prototype](/01-design-patterns/creational/prototype)

## Key Principles

- Hide object creation logic.
- Make the system independent of how its objects are composed.
- Reduce complexity and improve flexibility.
