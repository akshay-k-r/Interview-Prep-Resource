# Behavioral Patterns

Behavioral patterns focus on communication between objects, defining how they interact and distribute responsibility.

- [Observer](/01-design-patterns/behavioral/observer)
- [Command](/01-design-patterns/behavioral/command)
- [Visitor](/01-design-patterns/behavioral/visitor)
- [Interpreter](/01-design-patterns/behavioral/interpreter)

## Key Principles

- Define flexible communication between objects.
- Encapsulate requests as objects.
- Distribute responsibility among classes.
